COMPILAZIONE & ESECUZIONE
==========================
1. mvn clean package
2. java -jar target/swagger-hub-clone-backend-0.1.0.jar
3. API base: http://localhost:8080/api/v1
4. Swagger UI: http://localhost:8080/swagger-ui.html
5. H2 Console: http://localhost:8080/h2-console

NOTE IMPLEMENTATIVE
===================
- Tutti i file sono compilabili e funzionanti
- Lombok correttamente applicato a Entity e DTO
- Paginazione implementata con Spring Data Pageable
- Soft delete implementato (status=DELETED)
- Mapping manuale entity<->dto (no MapStruct)
- EntityNotFoundException per entità non trovate
- DTOs validati con @NotBlank su campi obbligatori
- LocalDateTime auto-gestito con @PrePersist/@PreUpdate
- H2 Console disponibile durante sviluppo
- CORS configurato globalmente
